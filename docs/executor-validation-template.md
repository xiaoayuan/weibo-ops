# 真实执行联调记录模板

## 使用方式

每遇到一个真实执行成功或失败样本，就复制下面模板填一份。

建议一个样本一段，不要混写。

我后续可以直接根据这份记录判断：

1. 是前端问题
2. 是 `apps/api` 路由问题
3. 是调度问题
4. 是执行器问题
5. 是微博平台返回差异
